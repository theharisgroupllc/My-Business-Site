# Everon Global Trades LLC Website

Modern Next.js retail storefront for **everonglobaltrades.com** with:

- Responsive premium UI
- Multi-category shop with filters
- Dynamic product pages
- Trust-focused homepage and business pages
- Tailwind CSS component-driven architecture

## Run locally

1. Install dependencies
   - `npm install`
2. Start development server
   - `npm run dev`
3. Open
   - `http://localhost:3000`
